よくあるご質問

FAQ

どんなサイトに対応していますか？

ATBB / REINS / イタンジBB に対応済み。いえらぶBB / いい生活スクエア も順次対応予定です。

アプリのインストールが必要ですか？

Chrome に拡張機能をインストールするだけ。業者サイトを開けば自動で「📥 帯付き」ボタンが表示されます。

自社の帯はどう登録するんですか？

拡張機能アイコンから設定画面を開き、帯画像を 1 回アップロードするだけ。担当者ごとの複数登録も可能です。

帯がない場合はどうなりますか？

ご安心ください。初回登録特典として、Claude AI と現役不動産プロがお客様専用の特製オリジナル帯 (¥10,000 相当) を無料デザインプレゼントしております。ヒアリングから 1 営業日で納品いたします。

業者サイトのログイン情報は送信されますか？

いいえ、一切送信されません。拡張機能はお使いの Chrome 上で完結する設計で、ID・パスワード・セッション情報は弊社サーバーを経由しません。

個人で契約できますか？

個人クレカで契約 OK、会社契約は不要です。店舗導入の場合は店舗プランがおトクです。

50 件を超えたらどうなりますか？

引き続き利用可。51 件目以降は 1 件あたり ¥10 が翌月初旬に加算請求されます。

個人プランと店舗プランの違いは？

機能は完全に同じ。違いは件数と料金です。個人は 50 件、店舗は 300 件まで込みです。

解約はいつでもできますか？

いつでもキャンセル可能。月の途中で解約しても月末まで利用可。日割り返金はありません。

Chrome 以外でも使えますか？

現時点では Chrome および Chromium ベースの Brave、Edge 等のみ対応。Safari / Firefox は未対応です。

業者サイトの仕様変更で動かなくなることは？

あります。発生時は速やかにアップデート対応し、Chrome Web Store 経由で自動配信します。

複数店舗・FC 運営です。エンタープライズ契約は？

info@obigae-go.com までご相談ください。

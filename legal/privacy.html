プライバシーポリシー

最終更新日: 2026年5月22日 / 適用開始日: 2026年5月22日

東京ベイハウス株式会社（以下「当社」といいます）は、当社が提供する Chrome 拡張機能サービス「帯替えGO‼︎」（以下「本サービス」といいます）における、ユーザーの個人情報の取扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます）を定めます。

1. データ取扱いの基本方針 (重要)

本サービスの中核機能（業者間流通サイト上の募集図面の取得・自社帯画像の焼き込み・PDF 生成）は、ユーザーの Chrome ブラウザ内（ローカル環境）で完結する設計となっています。具体的に：

  • 業者間流通サイト（ATBB / REINS / イタンジBB 等）のログイン情報・セッション情報を当社サーバーに送信することはありません。
  • 業者間流通サイトから取得した募集図面の PDF / 画像データを当社サーバーに送信することはありません。
  • ユーザーが登録した自社帯画像は、Chrome 拡張機能のローカルストレージ内にのみ保存されます。
  • AI / 機械学習による画像解析処理は一切行わず、外部の AI サービス（OpenAI / Anthropic / Google 等）に対するデータ送信もありません。

2. 収集する個人情報

当社サーバーが取得する情報は、アカウント認証・課金管理・お問い合わせ対応に必要な最小限の項目に限定されます。

3. 取得しない情報 (重要)

本サービスは、以下の情報を一切取得しません。

  • 業者間流通サイトのログインID・パスワード・セッショントークン
  • 業者間流通サイトから取得した募集図面の PDF / 画像データ
  • 募集図面に記載された物件情報（賃料・所在地・物件名・オーナー情報等）
  • ユーザーが Chrome 拡張機能内で登録した自社帯画像（ローカルストレージ内のみで管理）
  • 業者間流通サイト以外の Web 閲覧履歴・タブ情報

4. 利用目的

当社は、収集した個人情報を以下の目的で利用します。

  1. 本サービスの提供・運営
  2. ユーザーからのお問い合わせへの対応
  3. 利用料金の請求・決済
  4. サービスの品質改善・新機能開発
  5. 不正利用の検知・防止
  6. 法令に基づく場合の対応

5. 第三者提供

当社は、以下の場合を除き、ユーザーの同意を得ずに個人情報を第三者に提供しません。

  1. 法令に基づく場合
  2. 人の生命・身体・財産の保護のために必要がある場合
  3. 業務委託先への提供（決済処理・クラウドホスティング等。委託先には適切な監督を行います）
  4. 合併・事業譲渡等に伴う場合

ユーザーは、当社に対し、次の権利を行使することができます。

  2. 個人情報の訂正・追加・削除請求
  3. 個人情報の利用停止・消去請求
  4. 第三者提供の停止請求

これらの請求は、第10項記載の連絡先までお問い合わせください。

9. Cookie等の利用・アクセス解析

本サービスでは、認証維持・利便性向上のためにCookieおよびブラウザのlocalStorageを利用します。Cookie/localStorageを無効にすると、本サービスの一部機能が利用できなくなる場合があります。

また、利用状況の把握およびサービス改善のため、以下のアクセス解析ツールを使用しています。これらのツールはCookieを利用し、お客様の個人情報を含まない形で利用情報（ページ閲覧、クリック、スクロール、画面操作の記録等）を匿名で収集します。

  • Meta Pixel（提供: Meta Platforms, Inc.）— 広告の効果測定および配信最適化のため。Metaのプライバシーポリシー
  • Microsoft Clarity（提供: Microsoft Corporation）— ユーザーインターフェース改善のためのヒートマップ・セッション録画。個人を特定する情報（メールアドレス、パスワード等）は自動的にマスクされます。Microsoftのプライバシーステートメント

これらのツールが収集する情報は、上記の第三者によって匿名化された統計データとして処理されます。お客様はブラウザのCookie設定または各ツールのオプトアウト機能により、データ収集を拒否できます。

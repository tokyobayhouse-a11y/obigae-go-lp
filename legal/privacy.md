# プライバシーポリシー

最終更新日: 2026年5月8日（テンプレ初版・寺島さんレビュー待ち）

東京ベイハウス株式会社（以下「当社」といいます）は、当社が提供するクラウドサービス「見積りくん」（以下「本サービス」といいます）における、ユーザーの個人情報の取扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます）を定めます。

## 1. 収集する個人情報

当社は、本サービスの提供にあたり、以下の情報を収集します。

| 項目 | 取得方法 | 利用目的 |
|---|---|---|
| メールアドレス | 会員登録時 | アカウント識別・連絡 |
| パスワード | 会員登録時 | 認証（暗号化して保存） |
| クレジットカード情報 | 決済時 | 料金決済（**当社サーバーには保存せず、Stripe社で処理**） |
| 利用ログ | 自動収集 | サービス品質改善・不正検知 |
| IPアドレス・デバイス情報 | 自動収集 | 不正利用検知・セキュリティ確保 |

## 2. アップロードファイルの扱い

ユーザーが本サービスにアップロードした募集図面（PDF/画像）は、AI処理直後に削除し、当社のサーバーには永続保存しません。AI処理（Anthropic Claude API）に必要な範囲で一時的に Anthropic 社のサーバーに送信されますが、Anthropic 社のポリシーに従い、API経由のデータは学習に使用されません。

## 3. 抽出結果（見積データ）の扱い

ユーザーが保存した見積データは、ユーザー本人のみが閲覧可能で、他のユーザーや当社が業務上必要な場合を除き閲覧することはありません。データはSupabase（米国/シンガポールリージョン）のセキュアなデータベースに保存されます。

## 4. 利用目的

当社は、収集した個人情報を以下の目的で利用します。

1. 本サービスの提供・運営
2. ユーザーからのお問い合わせへの対応
3. 利用料金の請求・決済
4. サービスの品質改善・新機能開発
5. 不正利用の検知・防止
6. 法令に基づく場合の対応

## 5. 第三者提供

当社は、以下の場合を除き、ユーザーの同意を得ずに個人情報を第三者に提供しません。

1. 法令に基づく場合
2. 人の生命・身体・財産の保護のために必要がある場合
3. 業務委託先への提供（決済処理・クラウドホスティング等。委託先には適切な監督を行います）
4. 合併・事業譲渡等に伴う場合

## 6. 業務委託先

本サービスの運営にあたり、以下の事業者にデータ処理を委託しています。

| 委託先 | 用途 | データ保管地域 |
|---|---|---|
| Netlify, Inc. (米国) | サーバー・関数実行環境 | 米国 |
| Supabase, Inc. (米国) | 認証・データベース | シンガポール |
| Anthropic, PBC (米国) | AI抽出処理 | 米国（API経由・学習に使用されない） |
| Stripe, Inc. (米国) | 決済処理 | 米国（PCI DSS準拠） |
| OpenAI, OPC (米国) | 補助的画像処理（必要時） | 米国 |

これらの事業者はいずれも、相応のデータ保護水準を備えた事業者であり、当社は適切な契約・監督を行っています。

## 7. データの保管期間

| データ種別 | 保管期間 |
|---|---|
| アカウント情報 | 解約後30日で削除 |
| 見積データ | 解約後30日で削除 |
| 利用ログ | 36ヶ月（不正検知のため） |
| 決済情報 | Stripeの規定に従う（最大7年） |

## 8. ユーザーの権利

ユーザーは、当社に対し、次の権利を行使することができます。

1. 個人情報の開示請求
2. 個人情報の訂正・追加・削除請求
3. 個人情報の利用停止・消去請求
4. 第三者提供の停止請求

これらの請求は、第10項記載の連絡先までお問い合わせください。

## 9. Cookie等の利用・アクセス解析

本サービスでは、認証維持・利便性向上のためにCookieおよびブラウザのlocalStorageを利用します。Cookie/localStorageを無効にすると、本サービスの一部機能が利用できなくなる場合があります。

また、利用状況の把握およびサービス改善のため、以下のアクセス解析ツールを使用しています。これらのツールはCookieを利用し、お客様の個人情報を含まない形で利用情報（ページ閲覧、クリック、スクロール、画面操作の記録等）を匿名で収集します。

- **Meta Pixel**（提供: Meta Platforms, Inc.）— 広告の効果測定および配信最適化のため。[Metaのプライバシーポリシー](https://www.facebook.com/policy.php)
- **Microsoft Clarity**（提供: Microsoft Corporation）— ユーザーインターフェース改善のためのヒートマップ・セッション録画。個人を特定する情報（メールアドレス、パスワード等）は自動的にマスクされます。[Microsoftのプライバシーステートメント](https://privacy.microsoft.com/privacystatement)

これらのツールが収集する情報は、上記の第三者によって匿名化された統計データとして処理されます。お客様はブラウザのCookie設定または各ツールのオプトアウト機能により、データ収集を拒否できます。

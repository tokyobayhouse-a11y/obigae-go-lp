# 帯替えGO‼︎ LP用イラスト 生成指示書 (ChatGPT 用)

**最終更新**: 2026-05-22 (現行LPに合わせて改訂版)

## 0. このドキュメントの使い方

1. **ChatGPT を開く** (画像生成可能なプラン)
2. **このドキュメント全体をコピペ** して送る
3. ChatGPT に **「これを読んだら、Scene 1 から1枚ずつ生成してください」** とお願いする
4. 良ければ保存・微妙なら下記「修正フレーズ集」で再生成
5. 3枚揃ったら以下のファイル名で保存:
   - `scene1.png`
   - `scene2.png`
   - `scene3.png`
6. 3枚を `/Users/kichiqrio/Documents/Claude/obigae-go-lp/illust/` に置いて Claude に伝える

## 1. プロジェクト背景

- **製品名**: 帯替えGO‼︎ (おびがえゴー)
- **製品種別**: 賃貸不動産営業マン向けの Chrome 拡張機能
- **何をするツール?**: 業者間流通サイト (ATBB・REINS・イタンジBB) で募集図面 (マイソク) の "帯" (連絡先バナー) を 1クリックで自社の帯に差し替える
- **LP のトーン**: 業界内輪トーク + カジュアル親しみ (深刻すぎず、堅すぎず、業界の本音で語る)

---

## 2. ビジュアル・スタイル ガイドライン (全 Scene 共通)

| 項目 | 指定 |
|---|---|
| **スタイル** | 日本の現代アニメ・マンガ風 ビジネスイラスト (Modern Japanese anime/manga business illustration style) |
| **構図** | 横長 widescreen 2分割パネル (Left = Before / Right = After) |
| **サイズ** | 1536×1024 ピクセル (3:2 ワイドスクリーン) |
| **左パネルの色調** | 寒色 ─ ネイビー・グレー・暗めの青、ストレスフルな雰囲気 |
| **右パネルの色調** | 暖色 ─ オレンジ・暖色光、希望・安心感 |
| **キャラクター** | 日本の20-40代ビジネスマン (営業マン)、お客様夫妻 (男女ペア) |
| **服装** | スーツ・ワイシャツ・ネクタイ (営業マン)、カジュアルめのきれいめ (お客様) |
| **舞台** | 日本の不動産仲介オフィス、または接客スペース |
| **テキスト** | **絶対に入れない** (文字・ロゴ・看板・書類の文字も読めないようにぼかす) |

**ブランドカラー (再掲・厳守):**
- ネイビー: `#0d4d6b` (左パネルのトーンに反映)
- オレンジ: `#d97706` (右パネルのトーンに反映)
- これは姉妹製品「見積りGO‼︎」と統一の配色

## 3. Scene 1: 接客中の Before / After

**LP上の見出し**: 「接客中、お客様を待たせない」
**LP上のサブ**: 「5〜10枚の帯替え、お客様の目の前で『ちょっと待ってください...』のあの変な間、もう要りません」

### Left panel (Before, 寒色 / ネイビー基調)
- **場所**: 不動産屋のオフィス内、コピー機の前
- **人物**: 日本のサラリーマン (営業マン、30代)、ワイシャツ姿でストレスフルな表情
- **行動**: ハサミでマイソク(募集図面)の帯部分を切り取って、自社の帯シールをペタペタ貼っている
- **背景**: お客様夫妻 (男女) がやや離れた椅子で待っていて、時計を気にしている、退屈そう
- **小物**: コピー機、紙の束、ハサミ、シール、雑然としたデスク
- **雰囲気**: 焦り、気まずさ、お客様を待たせている罪悪感

### Right panel (After, 暖色 / オレンジ基調)
- **場所**: 同じオフィス、明るい接客スペース
- **人物**: 同じ営業マン、自信のある柔らかな笑顔
- **行動**: 綺麗に仕上がった募集図面 (自社の帯付き) を、お客様夫妻にスッと手渡している
- **背景**: 同じお客様夫妻、笑顔で図面を受け取り、興味深そうに見ている
- **小物**: デスクの上にノートPC (画面にブラウザらしき表示)、コーヒーカップ、整然とした空間
- **雰囲気**: スマート、プロフェッショナル、安心感、信頼

## 4. Scene 2: 反響対応で送る "1通" の Before / After

**LP上の見出し**: 「反響対応で"抜かれない"1通を送る」
**LP上のサブ**: 「希望物件を一気に『📥 帯付き』。エディタに並べて結合PDF化、メールやLINEで即送付。お客様の手元に残るのは自社の連絡先のみ」

### Left panel (Before, 寒色 / ネイビー基調)
- **場所**: 営業マンがオフィスのデスクで PC を使って、お客様にメールでマイソク (募集図面) を送ろうとしている
- **人物**: 困った表情の営業マン、PC画面を見ながらタイプしている
- **行動**: 営業マンが送ろうとしているマイソクには **他社の帯 (灰色の長方形)** がついたまま — 営業マンが気づいて困惑
- **対比**: 画面の別ウィンドウで、お客様夫妻 (送り先) が自宅で別のソファに座って、スマホ画面を見ながら **顔を寄せ合って "こそこそ" 話している** ニュアンス
- **小物**: PC画面、書類、マイソク、スマホ
- **雰囲気**: 営業マンが取り残された不安感、お客様が他社の方を向いてしまう匂わせ

### Right panel (After, 暖色 / オレンジ基調)
- **場所**: 同じデスク、明るく整った空間
- **人物**: 自信のある営業マン、PC画面でスムーズに送信操作中
- **行動**: 画面には3物件分のマイソクが綺麗に並んだ結合PDF、**全て自社の帯 (オレンジ系の統一感ある帯)** に差し替わっている
- **対比**: お客様夫妻がスマホで受け取って、笑顔で営業マンに連絡を取ろうとしている
- **小物**: PC画面の送信完了マーク、スマホ画面のメール受信通知、コーヒー、観葉植物
- **雰囲気**: 信頼、夫妻と営業マンが同じ方向を向いている、契約に向かう前向きな空気

## 5. Scene 3: 新人/ベテランの Before / After

**LP上の見出し**: 「新人もベテランも、"形から" 揃う」
**LP上のサブ**: 「営業に自信がない人には、第一印象を作る武器に。デキる人には、追客と接客の本質に時間を戻す道具に。店舗プランなら全員で品質を揃えられます」

### Left panel (Before, 寒色 / ネイビー基調)
- **場所**: 不動産屋のオフィス、新人のデスク
- **人物**: 若い新人営業マン (20代)、自信なさげ・うつむき気味
- **行動**: 手作業で雑に貼った帯シールがズレた、見栄えの悪いマイソクを手にして困った表情
- **背景**: 別のデスクでベテラン営業マン (40代) が電卓と紙束に埋もれている (彼も帯替えで時間を消費しているニュアンス)
- **小物**: ハサミ、シール、ヨレヨレのマイソク
- **雰囲気**: バラツキ、自信のなさ、品質ムラ、店舗全体が消耗している

### Right panel (After, 暖色 / オレンジ基調)
- **場所**: 同じオフィス、明るく整った空間
- **人物**: 同じ新人営業マンが、綺麗な帯付きマイソクを手に **自信に満ちた笑顔** で堂々と立っている
- **行動**: マイソクを手にして接客に向かおうとしている (またはお客様にお渡ししている)
- **背景**: ベテラン営業マンも余裕の表情で電話 (追客中)、お客様らしき人と笑顔で話している
- **小物**: ノートPC・スマホ・整然としたデスク
- **雰囲気**: 店舗全体が同じ品質で揃っている、新人もベテランも本来の営業活動に集中できている

## 7. 微妙な時の修正フレーズ集

- **左パネルをもっとストレス感出して**: 「LEFT panel をもっと暗く陰鬱な雰囲気にして、営業マンの表情をもっと焦った感じに」
- **右パネルをもっと希望的に**: 「RIGHT panel をもっと明るい暖色光に、営業マンとお客様の表情を満足感あるものに」
- **キャラクターが外国人っぽい**: 「キャラクターを **日本人らしい顔立ち・髪型** にして、日本のオフィスらしい背景にして」
- **テキストが入ってしまった**: 「画像内の文字・ロゴ・看板・読める書類の文字を **全て削除** して、抽象的な模様に置き換えて」
- **構図が縦になった**: 「**必ず横長 widescreen 1536×1024**, 縦長や正方形は使わないで」
- **色味が違う**: 「ベース色をネイビー #0d4d6b、右パネルにオレンジ #d97706 系の暖色を加えて」
- **スタイルが写真っぽい**: 「**現代の日本のアニメ・マンガ風のフラットなイラスト** にして、写真リアルな感じはやめて」
- **3枚で画風がバラつく**: 「Scene 1 と同じキャラクター画風・色トーンで揃えて (連作として一貫性を)」

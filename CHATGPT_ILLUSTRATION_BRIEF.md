# 帯替えGO‼︎ LP用イラスト 生成指示書 (ChatGPT 用)

## 0. このドキュメントの使い方

1. **ChatGPT を開く** (GPT-4o または GPT-5、画像生成機能がある有料プラン推奨)
2. **このドキュメント全体をコピペ** して送る
3. ChatGPT に **「これを読んだら、Scene 1 から1枚ずつ生成してください」** とお願いする
4. 出来上がった画像を見て:
   - 良ければ **保存** (右クリック → 名前を付けて保存)
   - 微妙なら **「ここを直して」と具体指示** して再生成
5. 3枚 (Scene 1 / 2 / 3) 揃ったら、以下のファイル名で保存:
   - `scene1.png` ── 接客中の Before/After
   - `scene2.png` ── 反響対応の Before/After
   - `scene3.png` ── 新人/ベテランの Before/After
6. 3枚を Claude に共有 (この対話に戻って画像を添付 or ファイルパス指定)

## 1. プロジェクト背景 (ChatGPT に共有する文脈)

- **製品名**: 帯替えGO‼︎ (おびがえゴー)
- **製品種別**: 賃貸不動産営業マン向けの Chrome 拡張機能
- **何をするツール?**: 業者間流通サイト (ATBB・REINS・イタンジBB) で募集図面の "帯" (連絡先バナー) を 1クリックで自社の帯に差し替える
- **LP の役割**: この拡張機能の紹介・販売ページ
- **LP のトーン**: 業界内輪トーク + カジュアル親しみ (深刻すぎず、堅すぎず、業界の本音で語る)

---

## 2. ビジュアル・スタイル ガイドライン (全 Scene 共通)

| 項目 | 指定 |
|---|---|
| **スタイル** | 日本の現代アニメ・マンガ風 ビジネスイラスト (Modern Japanese anime/manga business illustration style) |
| **構図** | 横長 widescreen 2分割パネル (Left = Before / Right = After) |
| **サイズ** | 1536×1024 ピクセル (16:9 widescreen) |
| **画面比率** | 横長ワイドスクリーン |
| **左パネルの色調** | 寒色 (Cool tones) ─ ネイビー・グレー・暗めの青、ストレスフルな雰囲気 |
| **右パネルの色調** | 暖色 (Warm tones) ─ 自然光・オレンジ・明るい黄色、希望・安心感 |
| **キャラクター** | 日本の20-40代ビジネスマン (営業マン)、お客様夫妻 (男女ペア) |
| **服装** | スーツ・ワイシャツ・ネクタイ (営業マン)、カジュアルめのきれいめ (お客様) |
| **舞台** | 日本の不動産仲介オフィス、または接客スペース |
| **テキスト** | **絶対に入れない** (文字・ロゴ・看板・書類の文字も読めないようにぼかす) |

**色のメイン:**
- ネイビー: `#1A2E4A`
- ブルー: `#4A9ECC`
- 暖色 (右パネル用): オレンジ系 `#F5A623` または `#FFB347`

## 3. Scene 1: 接客中の Before / After

**コンセプト**: お客様を待たせる "変な間" が消える瞬間

### Left panel (Before, 寒色)
- **場所**: 不動産屋のオフィス内、コピー機の前
- **人物**: 日本のサラリーマン (営業マン)、ワイシャツ姿でストレスフルな表情
- **行動**: ハサミでマイソク(募集図面)の帯部分を切り取って、自社の帯シールをペタペタ貼っている
- **背景**: お客様夫妻 (男女) がやや離れた椅子で待っていて、時計を気にしている、退屈そう
- **小物**: コピー機、紙の束、ハサミ、シール、雑然としたデスク
- **雰囲気**: 焦り、気まずさ、お客様を待たせている罪悪感

### Right panel (After, 暖色)
- **場所**: 同じオフィス、明るい接客スペース
- **人物**: 同じ営業マン、自信のある柔らかな笑顔
- **行動**: 綺麗に仕上がった募集図面(自社の帯付き)を、お客様夫妻にスッと手渡している
- **背景**: 同じお客様夫妻、笑顔で図面を受け取り、興味深そうに見ている
- **小物**: デスクの上にノートPC (画面にブラウザ表示、コーヒーカップ、整然とした空間
- **雰囲気**: スマート、プロフェッショナル、安心感、信頼

## 4. Scene 2: 反響対応の Before / After

**コンセプト**: 夫婦に「抜かれる」リスクが消える瞬間

### Left panel (Before, 寒色)
- **場所**: 不動産屋の店頭カウンター、または接客スペース
- **人物**: 不安そうな表情の営業マン、書類を手渡したばかり
- **行動**: 営業マンが手渡したマイソクを見ながら、お客様夫妻 (男女ペア) が **顔を寄せ合って "こそこそ" 話している**
- **背景**: マイソクの帯部分に他社の連絡先 (具体的な文字は読めないがロゴマーク的なもの)
- **小物**: マイソクの紙、ペン、書類
- **雰囲気**: 営業マンが取り残された感、お客様が他社のことを話している匂わせ、不穏

### Right panel (After, 暖色)
- **場所**: 同じ場所、明るい雰囲気
- **人物**: 自信のある営業マン、お客様夫妻と自然な距離感
- **行動**: 営業マンがタブレット (またはPC画面) を見せながら、3物件の図面をスムーズに紹介
- **背景**: 図面には自社の帯がしっかり (具体的な文字は読めないが帯部分が統一感ある色)
- **小物**: タブレット、コーヒー、観葉植物、明るい窓辺
- **雰囲気**: 信頼、夫妻と営業マンが同じ方向を向いている、契約に向かう前向きな空気

## 5. Scene 3: 新人/ベテランの Before / After

**コンセプト**: 営業マンに自信がない人は「形から」、できる人は「追客時間↑」、両方に効く

### Left panel (Before, 寒色)
- **場所**: 不動産屋のオフィス、新人のデスク
- **人物**: 若い新人営業マン (20代)、自信なさげ・うつむき気味
- **行動**: 手作業で雑に貼った帯シールがズレた、見栄えの悪いマイソクを手にして困った表情
- **背景**: 別のデスクでベテラン営業マン (40代) が電卓と紙束に埋もれている (彼も帯替えで時間を消費しているニュアンス)
- **小物**: ハサミ、シール、ヨレヨレのマイソク
- **雰囲気**: バラツキ、自信のなさ、品質ムラ、店舗全体が消耗している

### Right panel (After, 暖色)
- **場所**: 同じオフィス、明るく整った空間
- **人物**: 同じ新人営業マンが、綺麗な帯付きマイソクを手に **自信に満ちた笑顔** で堂々と立っている
- **行動**: マイソクを手にして接客に向かおうとしている (またはお客様にお渡ししている)
- **背景**: ベテラン営業マンも余裕の表情で電話 (追客中)、お客様らしき人と笑顔で話している
- **小物**: ノートPC・スマホ・整然としたデスク
- **雰囲気**: 店舗全体が同じ品質で揃っている、新人もベテランも本来の営業活動に集中できている

## 7. 微妙な時の修正フレーズ集 (コピペ用)

- **左パネルをもっとストレス感出して**: 「LEFT panel をもっと暗く陰鬱な雰囲気にして、営業マンの表情をもっと焦った感じに」
- **右パネルをもっと希望的に**: 「RIGHT panel をもっと明るい自然光に、営業マンとお客様の表情を満足感あるものに」
- **キャラクターが外国人っぽい**: 「キャラクターを **日本人らしい顔立ち・髪型** にして、日本のオフィスらしい背景にして」
- **テキストが入ってしまった**: 「画像内の文字・ロゴ・看板・読める書類の文字を **全て削除** して、抽象的な模様に置き換えて」
- **構図が縦になった**: 「**必ず横長 widescreen 1536x1024**, 縦長や正方形は使わないで」
- **色味が違う**: 「ベース色をネイビー #1A2E4A とブルー #4A9ECC、右パネルにオレンジ系の暖色を加えて」
- **スタイルが写真っぽい**: 「**現代の日本のアニメ・マンガ風のフラットなイラスト** にして、写真リアルな感じはやめて」

# 帯替えGO‼︎ ロゴ・アイコン 制作指示書 (ChatGPT 用)

## 0. このドキュメントの使い方

1. **ChatGPT を開く** (GPT-4o / GPT-5、画像生成可能な有料プラン)
2. **このドキュメント全体をコピペ** して送る
3. ChatGPT に **「これを読んだら、まずコンセプトA案を生成してください」** と依頼
4. 5パターン出してもらって寺島さんが選定
5. 選んだ案を 各サイズ (16×16 / 48×48 / 128×128 / 192×192 / 512×512 / LP用 横長 PNG) で書き出し
6. 揃ったら Claude に共有 → LP & Chrome 拡張に組み込み

## 3. ロゴのコンセプト方向性 (5案、いずれか / 複数を試す)

### 案 A: 「帯+GO!!」 文字主体 (mitsumori-go と最も統一)
- 「帯替え」を縦書き or 横書き、太いゴシック体
- 「GO‼︎」を斜めに走らせて躍動感
- 矢印モチーフを「替え」 or 「GO」 と組み合わせ
- 全体的にネイビー、「GO‼︎」だけオレンジで強調
- → mitsumori-go の「見積り + GO!! + 矢印」と双子のような関係

### 案 B: 「帯」シンボル + テキスト
- 横長のリボン (帯) が斜めに伸びるシンボル
- 帯の中に「GO‼︎」 or 矢印
- 横にテキスト「帯替えGO‼︎」
- 帯の躍動感 + シンプルなアイコン化のしやすさ

### 案 C: 「2本の帯+矢印」 = 帯替えの動作を表現
- 上に「グレーの古い帯」、下に「カラフルな新しい帯」
- 間に矢印 (上→下) で「替える」を表現
- → 「帯を替える」の動作が一目でわかる
- Chrome拡張の小さいアイコン (16px) でも視認しやすい

### 案 D: 「O」マーク (モノグラム) + 帯
- 「OBI」の頭文字「O」を大きく
- O の中に帯のストライプ模様
- O の右下に「GO‼︎」 オレンジで小さく
- 海外展開も視野に入れた英字ロゴ

### 案 E: 「カートゥーン親しみ系」
- 帯を持ったキャラクター (シルエット程度、顔は描かない)
- 「帯替えGO‼︎」テキスト
- 業界の現場感、親しみ
- Bランクのデザイン感、リスクあり

**寺島さんのおすすめ順 (推測)**: A → B → C (mitsumori-go との統一感が一番大事)。

## 5. デザイン要件 (絶対遵守)

### 5.1 視認性
- **16×16 px でも何のロゴかわかる**こと
- 小サイズでは細部の装飾を省略してOK (シンボル+α 程度)

### 5.2 セーフゾーン (アイコンの場合)
- ロゴ本体は **アイコン領域の 80% 以内** に収める (端から最低 10% の余白)
- 角丸の Chrome アイコン適用後でも切れないよう注意

### 5.3 背景
- アイコン: **透過 PNG (背景透明)** または **ベタ色背景**
- アイコンの背景がベタ色の場合は、ネイビー (#0d4d6b) を推奨

### 5.4 文字
- 「帯替えGO‼︎」のフォント: **太いゴシック・サンセリフ**
- 「GO‼︎」は若干斜め (12〜18度) で躍動感
- 「‼︎」は U+203C (DOUBLE EXCLAMATION MARK) + U+FE0E、または単に「!!」でも可

### 5.5 配色 (再掲・厳守)
- ベース: ネイビー #0d4d6b
- アクセント: オレンジ #d97706 (「GO‼︎」「矢印」など強調部分)
- 背景: 白 or ネイビー系の淡色

### 5.6 NG
- ❌ 写真・グラデーション過多
- ❌ 細かすぎる装飾 (16px で潰れる)
- ❌ 著作権ある既存ロゴ・キャラクターに似せる
- ❌ 不動産業界以外を連想させるアイコン (家・建物のシルエットも避ける、「帯」がテーマ)

## 7. 修正フレーズ集 (出力が微妙な時)

- **シンボルが複雑すぎる**: 「もっとシンプルに、16px でも識別できる形に」
- **テキストが読めない**: 「テキストは太いゴシック体で、字間を広めに」
- **色が違う**: 「ネイビー (#0d4d6b) とオレンジ (#d97706) のみ使用、他の色は使わないで」
- **「GO!!」が地味**: 「GO!! の斜め角度をもう少しつけて躍動感を、オレンジで強調」
- **不動産っぽくない**: 「帯 (リボン状の細長い長方形) のモチーフを必ず入れて」
- **過剰装飾**: 「Apple / Google 系の SaaS アプリアイコンのようなフラットでミニマルなデザインに」
- **見積りGO と似すぎ**: 「見積りGO とブランドファミリーだが、帯替えGO 独自の "帯" モチーフで差別化」

## 8. 完成チェックリスト

ロゴ案が決まったら以下を確認:

- [ ] **16×16 px に縮小しても識別可能**
- [ ] **配色がネイビー (#0d4d6b) + オレンジ (#d97706) のみ**
- [ ] **テキスト「帯替えGO‼︎」が読める** (大サイズでは)
- [ ] **「帯」のモチーフが入っている** (リボン状の長方形・横ストライプ等)
- [ ] **mitsumori-go と統一感はあるが、独自性も保持**
- [ ] **全6サイズ + 横長 LP 用が揃っている**
- [ ] **背景透過 PNG 推奨** (用途に応じて使い分け可)
